\section{Binomische Formeln}

\hfill \break
Binomische Formeln:
\begin{itemize}
    \item 1.Binomische Formel: $(a+b)^2 = a^2+2ab+b^2$
    \item 2.Binomische Formel: $(a-b)^2 =  a^2-2ab+b^2$
    \item 3.Binomische Formel: $(a+b)*(a-b) = a^2-b^2$
\end{itemize}
